# DiamondKata
Just some fun with programming exercise - draws diamond in the console output out of uppercase letters.

## How to run
### Example 1
```
.\DiamondKata.Console.exe A
A
```
### Example 2
```
.\DiamondKata.Console.exe B
_ A _
B _ B 
_ A _
```
### Example 3
```
.\DiamondKata.Console.exe C
_ _ A _ _
_ B _ B _
C _ _ _ C
_ B _ B _
_ _ A _ _
```
